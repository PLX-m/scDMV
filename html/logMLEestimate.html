logMLEestimate {scDMV}R Documentation

Parameter estimation with EM algorithm

Description

Parameter estimation with EM algorithm

Usage

logMLEestimate(
  binn,
  binx,
  intPi0,
  intPi1,
  intalph,
  intbeta,
  toll = 1e-05,
  iter = 1000
)

Arguments

binn

total reads

binx

methylation reads

intPi0

the first parameter: pi1

intPi1

the second parameter: pi2

intalph

the third parameter: alpha

intbeta

the fourth parameter: beta

toll

threshold. The default value is 0.00001

iter

iterations. The default value is 1000.

Details

The algorithm for estimating the four-parameter model is a random effects model with random coefficeint. The scDMV model use EM algorithm to estimate the four parameters.

Value

four parameters estimated by EM.
